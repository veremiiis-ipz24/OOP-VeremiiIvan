# Лабораторна робота №30

## Тема: Написання юніт-тестів з використанням xUnit

### Мета роботи

Ознайомитися з процесом написання юніт-тестів у середовищі .NET за допомогою фреймворку xUnit, навчитися використовувати різні типи перевірок (assertions) та параметризовані тести.

---

## Структура проєкту

* **lab30v2** — основний проєкт з бізнес-логікою
* **lab30v2.Tests** — тестовий проєкт (xUnit)

---

## Опис реалізації

У рамках роботи реалізовано клас **StringHelper**, який містить наступні методи:

* `Reverse(string input)` — перевертає рядок
* `IsPalindrome(string input)` — перевіряє, чи є рядок паліндромом
* `WordCount(string input)` — рахує кількість слів у рядку

---

## Тестування

Було написано **10 юніт-тестів**, які покривають:

* стандартні випадки використання
* граничні випадки (empty, null)
* некоректні дані
* різні варіанти вхідних значень

Використано:

* `[Fact]` — для окремих тестів
* `[Theory]` + `[InlineData]` — для параметризованих тестів

---

## Приклади перевірок

* перевірка реверсу рядка
* перевірка паліндромів (з пробілами, різним регістром)
* підрахунок слів у різних форматах рядків

---

## Запуск проєкту

Створення проєктів:

```
dotnet new console -n lab30v2
dotnet new xunit -n lab30v2.Tests
dotnet add lab30v2.Tests reference lab30v2
```

Запуск тестів:

```
dotnet test
```
